# ControlInventarios

Sistema de gestión de inventario desarrollado con **ASP.NET Core 10**, **Entity Framework Core** y **SQL Server** siguiendo una arquitectura por capas y buenas prácticas de desarrollo.

## Tecnologías utilizadas

- ASP.NET Core 10
- Entity Framework Core 10
- SQL Server 2025 Express
- Swagger / OpenAPI
- C#
- Clean Architecture

## Arquitectura

- Domain
- Application
- Infrastructure
- WebAPI
